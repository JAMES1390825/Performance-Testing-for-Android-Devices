# Android App 性能监控工具

Android 应用性能数据采集与分析工具。

## 功能

- **实时采集** - 采集应用 CPU、内存、FPS、卡顿率
- **启动测试** - 自动化测试冷启动、热启动时间
- **性能分析** - 生成统计报告
- **基线管理** - 版本间性能对比
- **可视化** - Web 界面实时查看性能曲线

## 采集指标

| 指标 | 说明 |
|------|------|
| `app_cpu_percent` | 应用 CPU 占用 (%) |
| `app_mem_kb` | 应用内存 PSS (KB) |
| `fps` | 帧率 |
| `jank_rate` | 卡顿率 (%) |

> 注：FPS 和卡顿率只有在应用有 UI 操作（滑动、动画等）时才会有数据，建议配合 UI 自动化测试使用。
